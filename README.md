## Starting up
(You might want to start a python environment)

### Requirements:
 - Python 3.13 (or similar)
 - pygame
 - import https://github.com/pvigier/perlin-numpy

Imports for the dependancies:
```bash
pip install pygame
```
```bash
pip install git+https://github.com/pvigier/perlin-numpy
```
