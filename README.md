## Starting up
(You might want to start a python environment)

```bash
python -m venv env
./env/Scripts/Activate.ps1
```

### Requirements:
 - Python 3.13 (or similar)
 - pygame
 - import https://github.com/pvigier/perlin-numpy

```bash
pip install -r requirements.txt
```

## Running

Just run
```bash
python main.py
```
